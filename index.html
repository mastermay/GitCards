GitCards

cards for GitHub users & repos

Prerequisites

jQuery

Usage

Setup

Include the script in your page

<link rel="stylesheet" type="text/css" href="stylesheets/gitcards.css">
<script type="text/javascript" src="javascripts/jquery-1.10.0.min.js"></script>
<script type="text/javascript" src="javascripts/gitcards.js"></script>

Activate

Add class="gitcards" in your element

use data-user="username" for a user card

use data-user="username" data-repo="reponame" for a repo card

Include the script in your page to activate GitCards ;-)

$(document).ready( function() {
    $('.gitcards').gitcards();
});

Example

<div class="gitcards" data-user="mastermay"></div>
<div class="gitcards" data-user="mastermay" data-repo="GitCards"></div>
